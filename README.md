This repository contains a collection of Practical Lab tasks performed in each module of CEHv13,  compiled for learning, revision, and exam preparation purposes.
The materials focus on hands-on ethical hacking concepts, tools, and workflows aligned with the CEH Practical / CEHv13 syllabus.
The PDFs in this repository cover practical labs related to:
1. Footprinting & Reconnaissance
2. Network Scanning & Enumeration
3. Vulnerability Analysis
4. System Hacking
5. Malware Threats
6. Sniffing & Network Attacks
7. Web Application Attacks
8. Wireless Network Attacks
9. Cryptography
10. Privilege Escalation Techniques
11. Post-Exploitation Basics

1. Nmap
2. Metasploit Framework
3. Hydra
4. John the Ripper
5. Hashcat
6. Netcat
7. Wireshark
8. Burp Suite
9. SQLMap
10. Responder
11. Windows & Linux command-line utilities
